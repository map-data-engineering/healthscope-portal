Data Update Frequency

Our datasets follow a structured update schedule to ensure accuracy and timeliness:

  • Core Facility Lists: Updated quarterly or when new data sources become available
  • Service Availability: Refreshed when MoH registries publish new exports
  • Coordinate Validation: Continuously verified against reference datasets
  • Quality Reports: Regenerated with each dataset version release

Data Harmonization Methodology

All source datasets are harmonized to a standard schema capturing: unique facility ID, official name, administrative hierarchy (country/region/district), facility type (hospital/clinic/health center), ownership (public/private/NGO), GPS coordinates, bed capacity, operational status, last verification date, and service availability indicators. Quality assessment includes duplicate detection, coordinate validation, admin hierarchy consistency, and temporal completeness checks.
